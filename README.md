# Python Programming

## How to Run

Make sure you have python installed on your system.

Run each task using the following commands:

```bash
python Task1.py

python Task2.py
```

Task 1:
This program takes an integer numbers as a user input and finds the factorial of that number using recursion and displays the output.

Task 2:
This program takes an integer numbers as a user input and finds the square root, logarithm and sine values of that number and displays the output.
